## Data Cleaning Hands-on Task

This project demonstrates basic data cleaning techniques using Python and Pandas.

## Tasks Performed
- Inserted missing values into the dataset
- Handled missing values using mean, median, and mode
- Removed duplicate rows
- Converted incorrect data types to numeric
- Saved the cleaned dataset as a new CSV file

## Files Included
- `task3_data_cleaning.ipynb` – Jupyter Notebook with data cleaning steps
- `rawdata.csv` – Original dataset
- `cleaned_data.csv` – Cleaned dataset after processing

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook
